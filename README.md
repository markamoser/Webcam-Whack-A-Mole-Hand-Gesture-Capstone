#Webcam Whack-A-Mole
## Overview
This is a game developed as a capstone project by four senior computer science students at the University of Nebraksa at Omaha. We are trying to create a proof-of-concept for using a hand gesture captrue by a webcam which can sense the depth of the hands moving towards and away from the sensor. Future applications will further develop the proof-of-concept if successful to create VR like controls without requiring individuals to hold. This is for accessibility features for people with disabilities or rehabilitating.

## Project Progress
We have successfully incorporated a single hand from the Media Pipe Hands library into unity and have built a functional version of Whack-a-Mole using the mouse as a controller. The next step is to make the Media Pipe Hands whack the moles. We split a second effort into getting ahead on the depth capture functionality. John and Hunter are working on depth capture. Mark and Ben are finishing the game and ironing out the two hands.
