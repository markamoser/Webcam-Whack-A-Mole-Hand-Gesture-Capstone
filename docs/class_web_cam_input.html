Detailed Description

Captures frames from a named webcam device and blits them into a RenderTexture, correcting for aspect ratio and vertical mirroring. If a static input texture is assigned in the Inspector, that texture is used instead and no webcam is opened. Attach to any GameObject and reference inputImageTexture from other scripts.
