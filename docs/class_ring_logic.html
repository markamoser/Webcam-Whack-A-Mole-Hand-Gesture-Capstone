Detailed Description

Follows the left hand marker point each frame and sets the public down boolean to true only when the left hand is both in a latched press state and still actively moving forward past the press velocity threshold. Used by Mole2DLogic to determine whether a whack attempt is valid. Attach to the LeftRing GameObject in the scene.
